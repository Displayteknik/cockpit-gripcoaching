# Bild-anti-klyschor — vad som ALDRIG ska genereras

Dessa är de visuella mönster som omedelbart signalerar "AI-genererat" eller "stockfoto-marknadsföring" och förstör trovärdigheten. Lägg dem i negativ-listan på VARJE bild-prompt.

## Top-10 mest dödande klyschor

1. **Handslag** — affärsmöte, två personer i kostym skakar hand. Dödar omedelbart.
2. **Glödlampa** — som symbol för idé. Generic AI-stock 101.
3. **Pussel-bitar** — som symbol för "lösning" eller "samarbete". Aldrig.
4. **Pilar uppåt-höger** — graf, ekonomisk framgång, ROI-visualisering.
5. **Kugghjul** — som symbol för "process". Funkar inte.
6. **Halv-leende affärsperson som tittar bortåt** — "professional" stock-pose.
7. **Person som håller telefon med leende** — fake natural moment.
8. **Glob med ljuspunkter** — global connectivity nonsens.
9. **Hjärna med nätverksledningar** — neural network-klichén.
10. **Kvinna som dricker kaffe vid bärbar dator i fönsterljus** — den absoluta freelancer-stocken.

## Branschspecifika klyschor att undvika

### Optometri / vården
- Person som håller upp glasögon mot kameran och ler
- Närbild på blå öga med teknisk overlay (matrix-style)
- Stetoskop på vit bakgrund
- Doctor-pose med stetoskop runt halsen och korslagda armar
- Patient-läkare som båda ler bredmunnigt vid undersökningsstol

### Coaching / ledarskap
- "Mentor som lägger hand på axeln"
- Klättra på berg-metafor med solnedgång
- Pil mot mål som splittras i flera mål
- Schackpjäs som symbol för strategi
- Cirkel av händer som möts i mitten
- Whiteboard med abstrakt strategi-diagram

### Tech / B2B
- Hologram-projektion ovanför handflata
- "Cyber" blå/lila gradient med data-strömmar
- Person som tittar på multipla skärmar samtidigt
- Hand som "trycker" på unikt knapp i tomma rummet
- Person framför kod på terminal-skärm

### Säljmotor / mysales-typ
- Säljpersonal med headset framför CRM-skärm
- "Funnel"-grafik med fallande prickar
- Pengar/sedlar som flygande element
- Trofé/medal-pose

## Subtilare AI-tells (inte klyschor men dödar)

1. **Symmetriska ansikten** — verkliga ansikten är asymmetriska
2. **Plastig hud** — för slät, för perfekt
3. **Felaktiga händer** — 6 fingrar, halva fingrar, smältande
4. **Kläd-mönster som loopar** — texturer som upprepar sig orealistiskt
5. **Bakgrund som inte håller** — AI hittar på arkitektur som inte funkar
6. **Skuggor som inte stämmer** — ljuset kommer från olika håll i samma bild
7. **Glasögon-reflexer** — alltid felaktiga
8. **Texter på skyltar/skärmar** — alltid trasiga, även på engelska
9. **Smyckens / klockor** — AI hittar på fantasy-smycken
10. **Tatueringar / pins** — slumpvis genererade, ofta meningslösa

## Mood-klyschor att undvika

- "Inspirerande person tittar mot soluppgång"
- "Hand som håller plantor med jord"
- "Tomma kontorsrum med stora fönster"
- "Brainstorming med post-its på glasvägg"
- "Diverse team som applåderar tillsammans"
- "Närbild på handen som håller penna mot anteckningsbok"

## Vad du gör istället

För **Ingela / Opticur**:
- Riktiga patient-momenterna (utan leende, naturligt)
- Närbild på ögon i naturligt ljus (utan teknik-overlay)
- Verkstaden/butiksmiljön i kvällsljus
- Hand som vrider mätinstrument (close-up med shallow depth)
- Tomma stolen i undersökningsrummet (efter avslutad behandling)

För **Carl-Fredrik / UGL-handledare**:
- Cirkel av människor i samtal (sett från sidan, inte ovanifrån)
- En person som lyssnar med eftertanke (close-up)
- Tomt rum med 12 stolar i cirkel
- Anteckningsbok med handskrivna ord (men inte svenska, eller helt textfri)
- Person som tittar ut genom fönster med eftertänksam blick

För **Hakan / Displayteknik**:
- Konkret display i butiksmiljö (inte studio)
- Hand som installerar/justerar
- Skylt som hänger / display som lyser i mörkt rum
- Detaljbild av displayens hörn / fästen (industrial, inte glansigt)

För **Hakan / GripCoaching**:
- Coaching-möte i avskalad miljö (kafé, kontor med få möbler)
- Två personer i samtal vid bord (sett över axeln)
- Människa skriver på papper (handen i fokus, ansikte ur fokus)
- Tom whiteboard med endast 2-3 viktiga ord

## Generell regel

**Tomma utrymmen, asymmetri, vardagsmiljöer, en enda fokuspunkt.**

Det är motsatsen till stockfoto. Det är vad människor faktiskt ser i sina liv.
