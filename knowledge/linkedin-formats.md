# LinkedIn-format — strukturmallar

## Textinlägg (200-400 ord)

```
[HOOK — 1 stark mening, gärna kort, gärna kontraintuitiv]
[Tom rad]
[Igenkänning — 2-3 meningar som speglar läsarens situation]
[Tom rad]
[Insikten — det nya perspektivet, gärna med ett konkret exempel eller siffra]
[Tom rad]
[Konkret tillämpning — 3-5 punkter eller en kort lista]
[Tom rad]
[Sammanfattning — en mening som förstärker insikten]
[Tom rad]
[Soft CTA — fråga eller inbjudan, inte påtryckning]
[Tom rad]
[Hashtags — max 3-4, alla relevanta]
```

## Karusell (6-10 slides, PDF-format)

- **Slide 1 (cover):** Hook + "Swipe →"
- **Slide 2:** Problem-ramning ("Här är varför detta är ett problem")
- **Slide 3-7:** En princip per slide. Stor rubrik (5-7 ord), 1-2 meningar body.
- **Slide 8 (sammanfattning):** "Sammanfattning: [3 bullets]"
- **Slide 9 (CTA):** "Spara, dela, eller DM:a [keyword] för [erbjudande]"

Designprincip: läsbarhet på telefon. Stor text. En idé per slide.

## Story-inlägg (story → insikt → tillämpning)

```
[Konkret scen — 2-3 meningar som sätter scenen, helst nyligen]
[Vändpunkt — vad som hände som ändrade tänket]
[Reflektion — vad du lärde dig]
[Generalisering — varför detta gäller läsaren]
[Tillämpning — vad de kan göra annorlunda nu]
[Soft CTA]
```

## Polls

```
[Hook + premiss på 2 rader]
Vilket är värst?
- Alternativ A
- Alternativ B
- Alternativ C
- Alternativ D (gärna en humoristisk)
[Kommentar i tråden där du analyserar resultaten efter 24h]
```

## Lead Magnet-inlägg

```
[Hook som lyfter ett konkret problem]
[Empati — visa att du förstår problemet på djupet]
[Lösning — beskriv vad guiden/checklistan ger på 3-5 punkter]
[Konkret resultat — vad de kommer kunna göra efter]
[CTA — "Kommentera 'GUIDE' så skickar jag den i DM"]
```

## Veckomix-mall (5 inlägg/vecka)

| Dag | Format | Trust-port | Posttyp |
|-----|--------|-----------|---------|
| Måndag | Textinlägg | KNOW | Insikt/principinlägg |
| Tisdag | Karusell | TRUST | Ramverk/hur-tänka |
| Onsdag | Textinlägg | LIKE | Personlig story |
| Torsdag | Textinlägg | TRY | Lead magnet eller case |
| Fredag | Textinlägg | LIKE | Reflektion/lärdom |

Varannan fredag → bytt mot direkt erbjudande (BUY-port). 20%-regeln.
